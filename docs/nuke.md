## `qdb nuke [CONFIG]`

removes all data and stops the current database

```
USAGE
  $ qdb nuke [CONFIG]

ARGUMENTS
  CONFIG  [default: .] directory of configuration file

OPTIONS
  -h, --help  show CLI help

EXAMPLES
  $ qdb nuke
  $ qdb nuke path/to/config
```
